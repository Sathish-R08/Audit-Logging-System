# Audit Logging System

A full-stack MERN application for managing and tracking audit logs.  
The project includes a React frontend and a Node.js + Express backend with MongoDB.

---

## Tech Stack

**Frontend**
- React
- JavaScript
- HTML, CSS

**Backend**
- Node.js
- Express.js
- MongoDB
- Mongoose
- dotenv

---

## Requirements

- Node.js (LTS)
- npm
- Git
- MongoDB (Local or Atlas)

##Check installations:

##1. Clone Repository
--git clone https://github.com/your-username/audit-logging-system.git
--cd audit-logging-system

##2. Backend Setup
--cd backend
--npm install
--Create .env file inside backend:

env
PORT=5000
MONGO_URI=your_mongodb_connection_string

node server.js
http://localhost:5000

3. Frontend Setup
cd frontend
npm install
npm start
http://localhost:3000
